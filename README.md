# 🪐 Dotlinetattu Handpoke & Traditional Tattoo Bali

Website profil profesional dan sistem manajemen konten (CMS) lokal berbasis MDX untuk **Dotlinetattu Tattoo Studio Bali**. Proyek ini dibangun menggunakan spesifikasi **Next.js App Router (2026)** dengan pendekatan desain *Brutalist Minimalist*.

## 🛠️ Tech Stack & Arsitektur

- **Framework:** Next.js (App Router) + TypeScript
- **Styling:** Tailwind CSS v4.0 + PostCSS
- **UI Components:** Radix UI Primitives via shadcn/ui
- **Content Engine:** Local MDX (`gray-matter` + `lib/mdx.ts`)
- **Optimization:** React Compiler (`reactCompiler: true`) untuk render otomatis tanpa memo manual
